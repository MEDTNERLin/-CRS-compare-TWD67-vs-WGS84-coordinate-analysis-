# 氣象站坐標系統分析

## 專案概述
本專案分析中央氣象署自動氣象站API (O-A0003-001)中的雙坐標系統，比較不同坐標參考系統的位置差異。

## 功能特色
- **雙坐標提取**: 解析API中的兩種坐標系統資料
- **距離計算**: 計算每個測站兩種坐標之間的距離差異
- **地圖視覺化**: 在同一張地圖上顯示兩種坐標位置
- **統計分析**: 分析坐標差距的分佈情況
- **圖表生成**: 生成距離分佈、累積概率等統計圖表

## 安裝與執行

### 1. 安裝依賴套件
```bash
pip install -r requirements.txt
```

### 2. 設定API金鑰
複製 `.env.example` 為 `.env` 並設定您的中央氣象署API金鑰：
```bash
cp .env.example .env
```
編輯 `.env` 檔案，將 `your_cwa_api_key_here` 替換為實際的API金鑰。

### 3. 執行分析
```bash
python coordinate_analysis.py
```

## 輸出檔案

### 分析結果
- `coordinate_analysis_*.csv` - 完整的測站坐標資料
- `coordinate_stats_*.json` - 統計摘要資料
- `coordinate_comparison_map_*.html` - 互動式地圖視覺化
- `coordinate_analysis_plot_*.png` - 統計分析圖表

### 地圖視覺化說明
- **藍色圓點**: 坐標系統1的位置
- **紅色圓點**: 坐標系統2的位置  
- **綠色連線**: 兩種坐標之間的距離差異（距離>10m時顯示）

### 統計圖表包含
1. 距離差異分佈直方圖
2. 累積分佈函數
3. 統計摘要表格
4. 距離分類圓餅圖

## 分析結果說明

### 坐標系統
氣象站API通常提供兩種坐標參考系統：
- **TWD97**: 台灣大地基準系統1997年版
- **WGS84**: 世界大地測量系統1984年版

### 距離計算
使用geopy套件的geodesic函數計算兩種坐標之間的大圓距離，單位為公尺。

### 統計指標
- 平均距離差異
- 中位數距離差異
- 標準差
- 最大/最小距離
- 百分位數分析
